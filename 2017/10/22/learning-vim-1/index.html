vim的基本操作（一）：移动

2017-10-22

这几天重新整了下windows下的开发环境，装了cmder又加了环境变量，现在方便用命令行操作了。
说到命令行下的操作，有的时候只是简单地修改下文件，而启动文本编辑器再打开文件很麻烦，就想到了vim。以后linux下编辑文件时也能节省学习成本。

##什么是Vim?##
Vim是由Unix Like系统内置的vi发展出来的文本编辑器。Vim具有代码补全、编译、错误跳转等方便编程的功能，而且使用鼠标的频率很少，完全能够脱离鼠标进行操作，让文本编辑更加迅速而不失强大。
有不少大佬将Vim配置成IDE使用，然而本弱鸡只是刚开始使用，在此记录一下Vim经常使用的基本操作。

##Vim的操作模式##
vim共有四种模式：分别是命令模式（Command mode），插入模式（Insert mode），正常模式 (Normal mode) ，可视模式 (Visual mode)。

  1. 正常模式（Normal mode）：
    在此状态下键盘的输入会被识别为命令。
  • 刚启动vim时，进入正常模式。
  • 正常模式一般用于浏览文件，也包括复制、粘贴、删除和翻页等操作，只有一些最基本的命令，要靠底线命令模式输入更多命令
  • 无论位于什么模式，按下ESC键都会进入正常模式。
  1. 输入模式（Insert mode）：
    在命令模式下输入i以进入输入模式。（还有其他的进入方法，之后说明）
    在输入模式中，可以使用：
  • 字符按键以及Shift组合，输入字符。
  • ENTER，回车键，换行。
  • BACKSPACE，退格键，删除光标前一个字符。
  • DEL，删除键，删除光标后一个字符。
  • 方向键，在文本中移动光标。
  • HOME/END，移动光标到行首/行尾。
  • Page Up/Page Down，上/下翻页。
  • Insert，切换光标为输入/替换模式，光标将变成竖线/下划线。
  • ESC，退出输入模式，切换到命令模式。
    可见输入模式与平常使用的输入方式相同。
  1. 命令模式（Command mode）：
    在正常模式下按:以进入命令模式。
  • 命令模式可以输入单个或多个字符的命令，可用的命令非常多。
  1. 可视模式（Visual mode）
    在正常模式下按下v,V,<Ctrl>+v,以进入可视模式。
    进入可视模式后，从当前光标位置开始，移动光标，之间的文本都会是选中状态。

##使用vim前应该知道的##

Q:如何生成一个随机字符串？
A:让新手退出vim。

想象下你高兴的用vim打开一个文档，然而你不仅不会操作vim，一顿乱敲删掉好多东西，更糟糕的是你发现自己不能退出vim！<Ctrl>+c或者<Ctrl>+z还有<Ctrl>+d都不能退出，这就非常尴尬了。
下面说一下如何保存文档以及退出vim。

  • :w [filename]：将编辑的数据写入硬盘档案中。
  • :w! [filename]：若文件属性为只读时，强制写入该档案。能不能写入，取决于对该档案的权限。
  • :q：退出vim。（若未将数据写入硬盘则会提示，不能退出）
  • :q！：强制退出vim，不保存档案。
  • :wq [filename]：储存后离开vim。
  • :wq! [filename]：强制储存后离开vim。
    其中可以用[filename]指定保存的文档。
    以上为常用的保存退出命令。

##移动操作##
vim使用h（向左）,j（向下）,k（向上）,l（向右）移动光标，也可以使用方向键移动。

  1. 行间移动
  • gg：移到文档第一行。
  • <SHIFT> + g：移到文档的最后一行。
  • <SHIFT> + h：移到当前屏的第一行。
  • <SHIFT> + l：移到当前屏的最后一行。
  • <SHIFT> + m：移到当前屏的中间行。
  • :行号： 移到对应行号的行。
  • <Ctrl> + d：向下翻页。
  • <Ctrl> + u：向上翻页。
  1. 行内移动
  • w：跳到下一个词的词首。
  • e：跳到当前词的词尾。
  • b：跳到当前词的词首，如果光标本身就在当前词的词首，则跳到前一词的词首。
  • <SHIFT> + ^：跳到行首（非空白字符）。
  • <SHIFT> + $：跳到行尾。
  • f + 字符：跳到下一个该字符出现的位置。之后按;可以继续跳向下一个位置。
  • <SHIFT> + f + 字符：跳到上一个该字符出现的位置。同上，之后按;可以继续跳向上一个位置。
